# 🧠 NumPy Quest: 90 Problems to Master Vectorization

Welcome to **NumPy Quest** — a progressively intense collection of 90 NumPy coding challenges designed to sharpen your numerical computing skills the right way: through pure problem-solving.

This isn’t a tutorial.
This isn’t a copy-paste exercise.
This is structured skill-building disguised as a game.

If you complete all 90 problems, you won’t just “know NumPy” — you’ll *think* in NumPy.

---

## 🎮 The Structure

The quest is divided into four escalating levels:

### 🟢 Level 1 – Foundations (Easy)

You build your base:

* Array creation and reshaping
* Indexing and slicing
* Basic statistics
* Simple masking and transformations

Clean fundamentals. Zero shortcuts.

---

### 🟡 Level 2 – Core Manipulation (Moderate)

Now things get interesting:

* Broadcasting
* Row/column operations
* Sorting and filtering
* Handling missing values
* Working with structured datasets

You begin replacing loops with logic.

---

### 🔵 Level 3 – Advanced Vectorization & Logic (Hard)

Loops become suspicious.

* Softmax and normalization
* One-hot encoding
* Rolling statistics
* Distance calculations
* K-means steps
* Monte Carlo simulations
* Image transformations

At this stage, performance starts to matter.

---

### 🔴 Level 4 – Extreme (Optimization & Linear Algebra)

Welcome to the deep end.

* PCA from scratch
* Logistic regression math
* Neural network layers
* Batched matrix operations
* Convolution using NumPy only
* Ridge regression
* Memory-efficient large-scale computations

This level tests mathematical maturity, efficiency, and dimensional awareness.

---

## 🧩 Rules of the Game

* Use **pure NumPy only**
* Avoid explicit Python loops whenever possible
* No external ML libraries
* Focus on vectorization
* Write clean, readable code
* Optimize when required

Some problems mimic real-world data science scenarios.
Some are constraint-based.
Some are designed to break inefficient solutions.

That’s intentional.
